Air Traffic Delay Prediction using AIML

  • Tech Stack: Python, AIML, Google Colab, React

Developed advanced models to improve air traffic prediction and control using a range of ML and AI algorithms, including CNN, RNN, neural networks, random forests, support vector machines, and regression models.

Preprocessed historical U.S. air traffic data (1990-2020) using Python libraries (Pandas, NumPy) in Google Colab and evaluated model performance with metrics like accuracy, precision, recall, F1-score, and AUC-ROC.

Integrated AI-driven solutions into existing systems to improve real-time decision-making.

Built a responsive web interface using HTML, CSS, JavaScript, and React, along with an API to incorporate real-time air traffic data for AI-driven delay predictions based on flight details.
